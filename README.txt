OSNOVNA SKOLA "9. maj" - Sastavci (prilagodjeni sajt)

Kako zameniti slike:
- images/zabrnjica.jpg  -> ubacite pravu fotografiju Zabrnjice (prepisati preko postojece)
- images/krajcinovici.jpg -> ubacite pravu fotografiju Krajcinovica

Kako dodati dokumenta:
- ubacite PDF u folder dokumenti/ (napravite ga ako ne postoji)
- u dokumenta.html zamenite linkove (href) na odgovarajuce fajlove, npr. href="dokumenti/godisnji-plan.pdf"

Mapa:
- u kontakt.html zamenite iframe src sa tacnim "Embed a map" linkom iz Google Maps (Share -> Embed a map).
